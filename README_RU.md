# HTML DOM Stats
[![License: GPLv3](https://img.shields.io/badge/license-GPL%20V3-green?style=plastic)](LICENSE)

Этот скрипт анализирует HTML-код страницы и отображает статистику в виде "древовидной" структуры: размер кода (в байтах), уровень вложенности и количество дочерних элементов. Он может быть полезен для задач по оптимизации фронтенда.
